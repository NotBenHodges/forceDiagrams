Force Diagrams!

Here are 3 datasets that can be represented by a force diagram.

1. Data 1

This dataset represents employees and the organizations they work for. A force diagram could useful for this dataset because nodes and edges could very well display the connections between workers in organizations.

2. Data 2

This dataset represents characters from the musical Les Miserables. Each character has a name a group assigned to it to organize which characters are related to each other. Here the edges on a force-directed graph could be useful in determining who is in contact with each other based on the number of edges.

3. Data 3

Here is a dataset that uses species as nodes and animals as subnodes. A diagram made from this dataset would be useful in determining what animals are in each species.
